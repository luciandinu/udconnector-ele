Enable copy/paste functionality between UD and Elementor

== Description ==

Enables copy/paste functionality between [UD](https://unremarkable.design) and Elementor.
UD Connector for Elementor will add a new **paste** option in contextual menu and new section menu, inside Elementor. 
To use it, just **copy** any resource from [UD](https://unremarkable.design) and **paste** it into Elementor.

== Installation ==

Instalation steps
* Make sure that you have Elementor installed and active
* Install and activate this plugin (UD Connector for Oxygen)
* Done. A new **paste** option will be available in the contextual menu and new section menu, inside Elementor. 
Just **copy** any resource from [UD](https://unremarkable.design) and **paste** it into Elementor.
